Fuse VJ

Limesonic is a collaboration of Tharit Firm Tothong and Qiu Yi Wu.

 

FuseVJ is an on-going project that aim to extend gestural movement of both performer and audience in performative context by utilizing the a captured live data feed from wearable sensors and mobile phones to generate algorithmic motion graphic. Ultimate goal of the project is to create a participatory visual performance that encourage participant and performer to move their body. While VJ curated the data feed and translate it into visual graphic that corresponds with the narrative of the performance. The project had recently featured in Maker Faire New York 2013.

 

Source code of the project can be view on github.
